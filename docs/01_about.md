# Smart Notes — описание проекта и границы

## 1. Название проекта
**Smart Notes** — кроссплатформенное интеллектуальное приложение для создания и классификации заметок.

## 2. Что делает приложение
Smart Notes позволяет пользователю быстро создавать короткие текстовые заметки и автоматически распределять их по категориям на основе содержимого текста.  
Приложение работает как PWA в браузере на настольных компьютерах и мобильных устройствах, сохраняет данные локально и помогает быстро находить нужные записи через фильтры.  
В первой версии интеллектуальность реализуется не через нейросеть, а через простой и понятный механизм классификации по ключевым словам.  
Такой подход делает приложение лёгким, быстрым и удобным для развёртывания через GitHub Pages.

## 3. Пользователь
Основной пользователь — студент или любой человек, которому нужно быстро фиксировать короткие заметки, списки дел и напоминания.  
Пользователь использует приложение, чтобы:
- не терять мелкие записи;
- быстро сортировать заметки без ручного выбора категории;
- открывать приложение на телефоне и компьютере без отдельной установки.

## 4. Функции первой версии
1. Создание текстовой заметки.
2. Автоматическое определение категории заметки по содержимому текста.
3. Просмотр списка всех заметок.
4. Фильтрация заметок по категориям.
5. Удаление заметки.
6. Сохранение заметок и настроек в локальном хранилище браузера.

## 5. Что не делаем
1. Регистрацию и авторизацию пользователей.
2. Синхронизацию заметок через облако или сервер.
3. Совместное редактирование несколькими пользователями.
4. Голосовой ввод и обработку аудио.
5. Использование полноценной ML-модели или внешнего AI API.

## 6. Платформа/стек (предварительно)
- **Платформа:** Desktop + Mobile через браузер (PWA)
- **Язык:** JavaScript
- **GUI-фреймворк:** HTML + CSS + JavaScript
- **Развёртывание:** GitHub Pages
- **Хранение данных:** localStorage
