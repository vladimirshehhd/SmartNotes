# Smart Notes — Тест-стратегия

## 1. Unit-тесты
1. Валидация пустой заметки.
2. Валидация заметки из одних пробелов.
3. Проверка ограничения длины 280 символов.
4. Классификация в категорию **Учёба**.
5. Классификация в категорию **Покупки**.
6. Классификация без совпадений в `other`.
7. Преобразование DTO -> Domain при неизвестной категории.
8. Форматирование даты.

## 2. Integration-тесты
1. Загрузка заметок из localStorage и маппинг в Domain.
2. Сохранение новой заметки и повторная загрузка.
3. Обработка битого JSON в localStorage.
4. Загрузка настроек по умолчанию при пустом хранилище.
5. Цепочка `ViewModel -> Service -> Storage`.
6. Фильтрация заметок по категориям.

## 3. E2E / Acceptance
1. Smoke-тест запуска приложения.
2. Добавление заметки через UI.
3. Отображение автоматической категории.
4. Удаление заметки.
5. Сохранение данных после перезагрузки страницы.
6. Ошибка при пустом вводе.
7. Переключение фильтра.

## 4. Инструменты
- **Unit/Integration:** Vitest
- **E2E:** Playwright
- **Ручная проверка:** Chrome/Edge DevTools
