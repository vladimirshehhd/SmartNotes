# Smart Notes — основные классы

### MainView
Отображает главный экран: форму добавления заметки, список заметок, фильтры и сообщения об ошибках.

### SettingsView
Показывает экран настроек и передаёт изменения параметров приложения.

### NotesViewModel
Связывает UI и бизнес-логику, хранит состояние экрана, вызывает сервисы и подготавливает данные для отображения.

## Application / Service

### NotesService
Зависит от `INotesStorage`, `IClassifier`, `IConfigProvider`.  
Выполняет создание, удаление, загрузку и фильтрацию заметок, а также их валидацию и классификацию.

### SettingsService
Зависит от `ISettingsStorage`, `IConfigProvider`.  
Загружает, валидирует и сохраняет настройки приложения.

### ClassifierService
Использует правила классификации и определяет категорию заметки по её тексту.

## Domain

### Note
Внутренняя модель заметки: текст, категория, дата создания, идентификатор.

### AppSettings
Внутренняя модель настроек приложения.

### NoteCategory
Перечисление допустимых категорий заметок.

### ClassificationRule
Правило сопоставления ключевых слов и категории.

## Infrastructure

### LocalStorageAdapter : INotesStorage, ISettingsStorage
Сохраняет и загружает заметки и настройки через `localStorage`.

### ConfigProvider : IConfigProvider
Предоставляет значения по умолчанию и конфигурационные параметры.

### BrowserCache
Хранит данные в памяти во время работы приложения для уменьшения числа повторных чтений.

### ServiceWorkerManager
Обеспечивает регистрацию service worker и базовую поддержку PWA.
