# Smart Notes

Smart Notes — это кроссплатформенное PWA-приложение для заметок с умной категоризацией по ключевым словам.

## Возможности

- создание заметок;
- автоматическая категоризация: Учёба / Покупки / Личное / Другое;
- фильтрация по категориям;
- поиск по тексту;
- удаление заметок;
- сохранение данных в `localStorage`;
- светлая и тёмная тема;
- установка как PWA;
- работа через GitHub Pages.

## Структура проекта

- `index.html` — основная страница;
- `styles.css` — стили интерфейса;
- `app.js` — логика приложения;
- `manifest.json` — настройки PWA;
- `service-worker.js` — offline-кеш;
- `icons/` — иконки приложения.

## Как запустить локально

### Вариант 1. Просто открыть
Можно открыть `index.html` в браузере, но часть PWA-функций может быть недоступна.

### Вариант 2. Через локальный сервер
Если установлен Python:

```bash
python -m http.server 8000
```

После этого открой:

```text
http://localhost:8000
```

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы проекта в корень репозитория.
3. Открой `Settings` -> `Pages`.
4. В блоке `Build and deployment` выбери:
   - `Source`: **Deploy from a branch**
   - `Branch`: **main**
   - `Folder`: **/ (root)**
5. Сохрани настройки.
6. Через короткое время GitHub выдаст ссылку на опубликованный сайт.

## Идея проекта

Smart Notes — учебный проект по дисциплине «Мультиплатформенное программирование интеллектуальных систем». Интеллектуальная часть реализована через автоматическую классификацию заметок по ключевым словам.
